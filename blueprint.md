# Project Blueprint

## Overview

This project is a web application that includes a lottery number generator with a theme switching feature, and a contact form powered by Formspree. It consists of an HTML file, a CSS file, and a JavaScript file.

## Features

- A user interface for the lottery number generator.
- A button to generate 6 unique random numbers between 1 and 45.
- The generated numbers are displayed on the screen.
- Dark mode/Light mode toggle functionality with local storage preference.
- **A simple contact form using Formspree for partnership inquiries.**
- The code is hosted on GitHub: https://github.com/withman37-hash/product1
